Links externos

A principal características da HTML é a possibilidade criar hiperlinks, proporcionando ao usuário consultar outros sites e documentos.

A forma mais facil de criar um link é usando a tag a e com o atributo href, que é obrigatório pois o valor que ele recebe é o endereço de destino.

Link para o <a href="https://youtube.com"> YouTube</a>

Porém, usando somento o atirbuto href, o novo site sobrepõe o atual. Isso não é bom quando se quer prender a atenção do leitor ao seu site. O atributo target com o valor _blank é usado para fazer com que o link abra em uma nova aba. Nao é uma regra, mas geralmente o target é usado para links de outros sites.

Link para o <a href="https://youtube.com" target="_blank" rel="external"> YouTube</a> em uma nova aba.

O atributo rel informa a relação do elemento com o documento, no caso, um link para um site externo, e geralmente é usado para motores de busca.
