Tabelas

Existe um conjunto de tags no HTML que permitem estruturar dados em duas dimensões no formato de uma tabela, na qual as informações ficam contidas em células no formato linhas e colunas.

  • <table>
  • <caption>
  • <td>
  • <tr>
  • <th>
  • <col>
  • <colgroup>
  • <thead>
  • <tbody>
  • <tfoot>

Todas as tags que compõem a estrutura de tabela têm tag de fechamento.

Estruturando de forma básica e sem atributos, a tabela não terá delimitações visíveis, nem cor de fundo e originalmente eram usados os atributos para definir as cores. Porém esse recurso está obsoleto, e hoje em dia as cores são configuradas usando as CSS.

Colunas - <col>

Como as tabelas são compostas basicamente de linhas e células de dados, a tag <col> informa ao navegador quais são as colunas e define suas propriedades, sendo apenas uma forma de referenciá-las, já que não exite tag para definir uma coluna, como a tag <tr> que define uma linha. A tag col também permite estilizar as colunas usando as CSS.

Estrutura da tabela - <table>

As tags <table> e </table> delimitam a estrutura da tabela.Todas as outras tags que formam a tabela são filhas da tag <table>.

Fluxo do conteúdo da tabela

Nesta ordem:

  1. um elemento <caption> opcional,
  2. zero ou mais elementos <colgroup>,
  3. um elemento <thead> opcional,
  4. uma das duas alternativas:
    • um elemento <tfoot> seguido por,
      • zero ou mais elementos <tbory>,
      • ou um ou mais elementos <tr>,
    • uma segunda alternativa seguida por um elemento <tfoot opcional
      • zero ou mais elementos <tbory>,
      • ou um ou mais elementos <tr>,

Legenda - <caption>

Define uma legenda para a tabela. Este elemento deve vir logo após o elemento <table>.

Células de dados - <td>

Cada célula é definida pelas tags <td> e </td> que contém os dados em seu contúdo.

A sigla td é do inglês table data.

Exemplo

As células de dados são containers da tabela que podem ter varias outras tags em seu conteúdo.
Antes das CSS, era muito comum serem usados tabelas para estilizar os sites.

Atributos

Atualmente, além dos atributos globais, três outros atributos podem ser usados:

colspan e rowspan
Fazem com que as células ocupem o espaço de outras células tanto na linha (rowspan) quanto na coluna (cowspan). Recebem valores inteiros positivos que configuram quantas linhas/colunas a célula vai ocupar. Colspan permite até 1000 colunas e rowspan permite até 65534 linhas, acima disso o valor é reduzido ao valor máximo. Além disso, definindo 0 ao rowspan, a célula será estendida ate o final da sessão da tabela que a contém.
headers
Recebe uma lista de strings separadas por vírgulas que correspondem ao atributo id de tags <th>.

Linha da tabela - <tr>

Define uma linha de células de dados na tabela. Pode conter também uma combinação de <td> (células de dados) e <th> (células de cabeçalho).

O número de linhas permitidas em uma tabela é ilimitado. Mas é uma boa pratica ter o mesmo número de espaços de células em todas as linhas, mesmo que não seja obrigatório pois algumas linhas levam menos células que outras, desde que a tabela esteja bem estruturada com rolspan e colspan.

A sigla tr é do ingles table row.

Cabeçalho da tabela - <th>

Funciona como um elemento <td> uma célula como um cabeçalho. Por padrão o texto contido neste elemento é apresentado em negrito e centralizado.

Além dos atributos colspan, rowspan e headers, o elemento ainda pode receber mais dois atributos:

abbr
Recebe um texto descrevendo o conteudo do cabeçalho da tabela.
scope
Possui quatro valores disponíveis (col, colgroup, row e rowgroup) que identificam se o cabeçalho é um cabeçalho da coluna, de um grupo de colunas, da linha ou de um grupo de linhas respectivamente.
